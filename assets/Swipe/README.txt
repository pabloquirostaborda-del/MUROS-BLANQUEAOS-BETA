README

Mapa interactivo sobre censura y memoria urbana en Medellín.

Archivos:
- index.html: estructura del mapa
- style.css: diseño visual
- script.js: lógica interactiva
- murales.geojson: base de datos georreferenciada

Para visualizar:
1. abrir la carpeta en Visual Studio Code
2. usar la extensión Live Server
3. abrir index.html
